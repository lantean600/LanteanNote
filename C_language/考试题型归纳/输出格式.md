# 总结

## 一、整型输出格式

| 格式符 | 含义 | 示例 `printf("%?d", 123)` | 输出 |
|--------|------|---------------------------|------|
| `%d` | 有符号十进制整数 | `%d` | `123` |
| `%i` | 同 `%d`（有符号十进制） | `%i` | `123` |
| `%u` | 无符号十进制整数 | `%u` | `123` |
| `%o` | 无符号八进制整数 | `%o` | `173` |
| `%x` | 无符号十六进制（小写） | `%x` | `7b` |
| `%X` | 无符号十六进制（大写） | `%X` | `7B` |
| `%ld` | 长整型十进制 | `%ld`, 123L | `123` |
| `%lld` | 长长整型十进制 | `%lld`, 123LL | `123` |
| `%lu` | 无符号长整型 | `%lu`, 123UL | `123` |

---

## 二、浮点型输出格式

| 格式符 | 含义 | 示例 `printf("%?f", 3.14159)` | 输出 |
|--------|------|-------------------------------|------|
| `%f` | 小数形式浮点数（默认6位小数） | `%f` | `3.141590` |
| `%e` | 指数形式（小写 e） | `%e` | `3.141590e+00` |
| `%E` | 指数形式（大写 E） | `%E` | `3.141590E+00` |
| `%g` | 自动选择 `%f` 或 `%e`（更简洁） | `%g` | `3.14159` |
| `%G` | 自动选择 `%f` 或 `%E`（大写） | `%G` | `3.14159` |
| `%lf` | `double` 类型（`printf` 中与 `%f` 相同） | `%lf` | `3.141590` |
| `%Lf` | `long double` 类型 | `%Lf`, 3.14159L | `3.141590` |

---

## 三、字符与字符串输出格式

| 格式符 | 含义 | 示例 | 输出 |
|--------|------|------|------|
| `%c` | 单个字符 | `%c`, 'A' | `A` |
| `%s` | 字符串 | `%s`, "hello" | `hello` |
| `%p` | 指针地址（十六进制） | `%p`, &x | `0x7fff...` |

---

## 四、格式修饰符（与格式符配合使用）

### 1. **宽度与精度**
```c
printf("%10d", 123);      // 宽度10，右对齐: "       123"
printf("%-10d", 123);     // 宽度10，左对齐: "123       "
printf("%010d", 123);     // 宽度10，补零: "0000000123"
printf("%.2f", 3.14159);  // 精度2位小数: "3.14"
printf("%8.2f", 3.14159); // 宽度8，精度2: "    3.14"
printf("%.4s", "hello");  // 字符串截取4字符: "hell"
```

### 2. **长度修饰符**
```c
%hd     // short int
%hu     // unsigned short
%ld     // long int
%lu     // unsigned long
%lld    // long long int
%llu    // unsigned long long
%Lf     // long double
%z      // size_t (C99)
%t      // ptrdiff_t (C99)
```

### 3. **标志修饰符**
```c
%-      // 左对齐
%+      // 显示正负号
%空格   // 正数前留空格
%#      // 显示进制前缀（0, 0x, 0X）
%0      // 用零填充宽度
```

## 五、特殊格式输出

### 1. **百分号本身**
```c
printf("%%");     // 输出: %
```

### 2. **转义字符输出**
```c
printf("换行:\n");   // \n 换行
printf("制表:\t");   // \t 水平制表
printf("双引号:\""); // \" 双引号
printf("反斜杠:\\"); // \\ 反斜杠
```

### 3. **常见转义字符**
| 转义序列 | 含义 | ASCII码 |
|----------|------|---------|
| `\n` | 换行 (LF) | 10 |
| `\r` | 回车 (CR) | 13 |
| `\t` | 水平制表 | 9 |
| `\v` | 垂直制表 | 11 |
| `\b` | 退格 | 8 |
| `\f` | 换页 | 12 |
| `\a` | 响铃 | 7 |
| `\\` | 反斜杠 | 92 |
| `\'` | 单引号 | 39 |
| `\"` | 双引号 | 34 |
| `\?` | 问号 | 63 |
| `\0` | 空字符 | 0 |
| `\xhh` | 十六进制字符 | 如 `\x41` = 'A' |
| `\ooo` | 八进制字符 | 如 `\101` = 'A' |

## 六、输入格式（`scanf`）对应关系

| 格式符 | 含义 | 变量类型 |
|--------|------|----------|
| `%d` | 有符号十进制整数 | `int*` |
| `%i` | 自动识别进制整数 | `int*` |
| `%u` | 无符号十进制整数 | `unsigned int*` |
| `%o` | 八进制整数 | `unsigned int*` |
| `%x` | 十六进制整数 | `unsigned int*` |
| `%f` | `float` 浮点数 | `float*` |
| `%lf` | `double` 浮点数 | `double*` |
| `%Lf` | `long double` 浮点数 | `long double*` |
| `%c` | 字符 | `char*` |
| `%s` | 字符串（无空格） | `char[]` |
| `%p` | 指针地址 | `void**` |
| `%[]` | 扫描字符集合 | `char[]` |
| `%%` | 匹配百分号 | - |

**特别注意**：
```c
float f;
scanf("%f", &f);      // float 用 %f

double d;
scanf("%lf", &d);     // double 用 %lf (与printf不同!)
```

## 七、考试常见输出陷阱

### 1. **整数与浮点数混合**
```c
printf("%d", 3.14);    // 错误！输出乱码
printf("%f", 5);       // 错误！输出0.000000或乱码
```

### 2. **精度截断与四舍五入**
```c
printf("%.2f", 3.145);   // 输出 3.15（四舍五入）
printf("%.2f", 3.144);   // 输出 3.14（四舍五入）
```

### 3. **宽度不足时**
```c
printf("%2d", 12345);    // 宽度2但实际5位 → 完整输出 "12345"
```

### 4. **八进制/十六进制前缀显示**
```c
printf("%#o", 10);      // 输出 012
printf("%#x", 10);      // 输出 0xa
printf("%#X", 10);      // 输出 0XA
```
