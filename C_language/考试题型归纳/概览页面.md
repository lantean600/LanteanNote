# 考试说明

### 考试题型与分值

- **单项选择题**：40题，共40分  
- **填空题**：10题，共10分  
- **判断题**：10题，共10分  
- **程序填空题**：4题，共20分  
- **程序设计题**：2题，共20分  

### 考试时间  
100分钟  

### 考试范围  
- 第1章 至 第6章  
- 指针部分不考二级指针  
- 第7章 结构体和自定义类型（仅涉及基础内容）  
- 第7章、第8章不涉及程序设计题
